# memo-application 
커피 한 잔 마시며 끝내는 Vue.js 를 학습하며 실습한 내용입니다.  

현재 `master` 브랜치는  
- 상태 관리 라이브러리인 Vuex를 연동하고
- Back-end API를 연동하였으며, 
- 코드를 리팩토링한 내용을 정리했습니다.
